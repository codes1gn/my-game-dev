> 状态：讨论中（2026-06-02）

## 一、核心问题

当前游戏的选择系统是 **规则式分支树**，所有路径写死，选择只是装饰。
这和roguelike的核心体验矛盾——roguelike需要 **随机性 + 构筑成长 + 每局不同**。

**目标：用"模拟计算"代替"脚本编写"。**

---

## 二、案件 = 隐藏数据模型（而非对话树）

每个案件在生成时，底层有隐藏变量：

```
案件数据 {
  真凶: NPC_ID
  动机: 仇杀 | 财杀 | 情杀 | 灭口
  作案手法: { 凶器, 手段, 伪装方式 }
  时间线: [ { 人物, 时间, 地点, 行为 } ]
  物证列表: [ { 名称, 位置, 发现难度, 关联真相 } ]
  证人: [ { 姓名, 可信度, 知道什么, 说谎点 } ]
  干扰信息: [ { 类型, 误导方向 } ]
}
```

这些变量从 **案件素材库** 随机组合生成。这就是roguelike的随机性来源。

---

## 三、调查 = 信息发现过程（属性决定信息量）

| 属性 | 调查中的作用 |
|------|------------|
| 观察力 | 现场能发现多少物证（5→60%，10→90%） |
| 审讯力 | 证人暴露多少信息 / 嫌疑人露出破绽 |
| 鉴定学 | 法医报告、技术鉴定结果的详细程度 |
| 心理学 | 犯罪心理侧写准确度 |
| 体能 | 调查时间预算（每天可用小时数） |
| 魅力 | NPC合作意愿、线人情报质量 |

**关键：不是"选对了走好路"，而是"你的能力决定你能获得多少信息"。**

---

## 四、推理 = 拼图式，不是选择题

- 玩家把收集到的证据 **自己连成推理链**
- 例：拖拽"第三人脚印" + "水果刀无凶手指纹" → 得出"有人栽赃"
- 游戏引擎检查推理链 vs 案件真相 → 给分
- 部分正确也给部分分数

## 五、天赋树 / 学习树

```
学习树
├── 观察系
│   ├── [基础] 细节之眼 → 观察+2
│   ├── [进阶] 微表情识别 → 审讯中自动标记谎言
│   └── [★金色] 全局视野 → 进入现场瞬间显示所有隐藏线索位置
├── 审讯系
│   ├── [基础] 压迫审讯 → 审讯+2
│   ├── [进阶] 心理画像 → 对话中显示嫌疑人情绪条
│   └── [★金色] 读心术 → 审讯时50%概率直接获得关键信息
├── 鉴定系
│   ├── [基础] 基础鉴定 → 鉴定+2
│   ├── [进阶] 交叉比对 → 两条证据可以合成新线索
│   └── [★金色] 一眼鉴真 → 法医报告直接标注死因和凶器
├── 体能系
│   ├── [基础] 耐力训练 → 每天可用时间+1小时
│   ├── [进阶] 追击本能 → 解锁"追踪嫌疑人"行动
│   └── [★金色] 铁人 → 调查不消耗时间（限1次/案件）
└── 社交系
    ├── [基础] 人脉网络 → 魅力+2
    ├── [进阶] 线人系统 → 每案获得1条免费情报
    └── [★金色] 权力之言 → 任意NPC强制说出全部真话
```

## 六、道具系统

| 道具 | 效果 | 获取方式 |
|------|------|---------|
| 高倍放大镜 | 现场解锁1条隐藏物证 | 系统商店 |
| 录音笔 | 审讯可回放+自动标记矛盾 | 学习树解锁 |
| 紫外灯 | 检查隐藏血迹/体液 | 系统商店 |
| 黑客U盘 | 解锁电子设备内容 | 学习树金色 |
| 推理直觉（消耗品）| 提示一条关键连接 | 系统点数兑换 |
| 权威传唤令 | 强制某证人到场 | 声望等级奖励 |
| 假身份证 | 伪装身份接近嫌疑人 | 学习树解锁 |

## 七、显式反馈机制（爽感来源）

**属性检定成功：**
- 金色提示框：`【观察力检定成功！】发现了隐藏线索：沙发垫下的手机`
- 专属音效

**技能触发：**
- 进阶技能：如"微表情识别" → 嫌疑人头像旁出现红色"谎言"标记
- 金色技能：全屏闪金光 + 大字 `【★全局视野★】所有隐藏线索已标记！`

**道具使用：**
- 紫外灯 → 现场画面切换紫外线模式，血迹发光出现
- 推理直觉 → 推理板上两条证据之间闪烁连线

**结案评分：**
- "本次案件发现 7/12 条线索，触发 3 个技能，使用 2 个道具"
- 对比："如果有【★金色·一眼鉴真】，可以额外发现 2 条线索"

## 八、Roguelike循环

```
接案 → 调查（属性/道具决定信息量）→ 推理（信息量决定难度）→ 结案评分
                                                                  ↓
                                                      获得系统点数 → 天赋树升级
                                                                  ↓
                                                      下一案更强 → 更高级案件
```

**时间预算系统：**
- 每案一天 = 8小时
- 每个调查行动消耗时间（去现场1h、审讯嫌疑人2h、等法医报告3h）
- 玩家自己决定在有限时间内做什么
- 时间用完 → 强制进入推理阶段
- 信息量 = 你的推理拼图碎片数
