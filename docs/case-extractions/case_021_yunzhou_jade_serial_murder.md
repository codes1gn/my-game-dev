## 三、核心侦查技巧

1. **翡翠鉴定+原石溯源**：从翡翠手镯追溯到雨落村祠堂→毛坯料来源→鬼市掮客
2. **鬼市探查**：明城鬼市（旧货市场+地下交易）的灰色地带调查
3. **DNA亲子鉴定反转**：郝震伦与姜名甫的父子关系→解锁姜名甫杀人动机
4. **无人机搜索热带雨林**：现代科技vs原始丛林→找到月清藏身草屋
5. **村民集体保护**：井芒村全村保护月清→"神医姐姐"→童言无忌泄露
6. **嫌疑人当面服毒**：月清精心策划的自杀——录完口供立即服毒→保护安银芝

## 四、深层主题

- **灰色正义**：郝震伦四人杀人夺宝→被月清借刀杀人→谁是正义？
- **以身饲虎**：安银芝为爱卧底、月清为义自杀→女性的极端牺牲
- **江洋大盗的浪漫**：安银木劫富济贫→受害者都活该→现代侠盗
- **法律的无奈**：陈益明知安银芝参与→无证据定罪→"认识也无所谓"
- **刑警的自我怀疑**："是我逼月清服毒的吗？"→查案的代价
- **尿毒症的绝望**：弟弟能治好→原石被夺→一切归零

## 五、游戏设计亮点

1. **鬼市探索系统**：旧货市场+地下交易+掮客NPC→灰色信息获取
2. **翡翠鉴定技能**：玩家需要识别翡翠品质、追溯来源
3. **热带雨林探索**：无人机搜索+丛林穿越+危险环境
4. **借刀杀人推理**：识破"自愿杀人"背后的教唆链条

6. **嫌疑人自杀事件**：紧急处理+后果承担+心理影响
