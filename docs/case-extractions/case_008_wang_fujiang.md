# 案件8：王福江案 / 头孢饮酒猝死+抛尸案

## 章节范围：第162-164章

## 案件类型标签
`意外死亡` `药物反应致死` `侮辱尸体` `乡镇案件` `短案过渡` `医学知识型`

## 案件概述
乡镇野地发现一具被野狗啃食的尸体。死者王福江（52岁农民）因服用头孢类药物后饮酒导致双硫仑样反应猝死。其酒友马承运因害怕赔偿，用三轮车将尸体抛至荒野，构成侮辱尸体罪。非谋杀案，但涉及法医学和农村社会问题。

---

## 一、因果链分析

### 背景
- 王福江：52岁农民，三个女儿一个儿子（高中），经济压力大
- 马承运：54岁杂货铺老板，王福江酒友
- 王福江因细菌性肺炎正在服用头孢类药物

### 事件链
1. 王福江因儿子未来花费（大学+结婚）心情郁闷
2. 晚饭后服用头孢药物
3. 外出散步，路过商店买酒，去马承运杂货铺喝酒
4. 大量饮酒→双硫仑样反应→猝死
5. 马承运恐惧赔偿责任→深夜用三轮车抛尸荒野
6. 尸体被野狗啃食→次日被发现→报警

### 侦查过程
- 法医尸检确定死因：双硫仑样反应（头孢+酒精）
- 监控追踪：三轮车载尸画面
- 走访确认：马承运与王福江无仇怨
- 结案：马承运以侮辱尸体罪处理
