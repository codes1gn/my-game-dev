## 一、完整因果链

### 1. 背景关系网

```
关系类型：熟人聚会中的多重矛盾网

陈益 ←(大学旧识/相互看不惯)→ 王立华(死者)
姜凡磊 ←(曾追求丁姿)→ 丁姿 ←(男女朋友)→ 王立华
王立华 ←(公司合伙人/股东, 王立华>50%控股)→ 袁利豪
许灿 ←(家族酒业竞争)→ 王立华
袁利豪 ←(秘密情人, 3个月前开始)→ 丁姿

关键背景：
- 许灿：聚会组织者，别墅主人，家里做酒业
- 王立华：富二代，自私狂妄占有欲强，有暴力倾向
- 丁姿：王立华女朋友，被打骂但不敢分手
- 袁利豪：白手起家，第二股东，表面成功人士
- 姜凡磊：陈益好友，曾追求丁姿
- 王立华有鼻中隔偏曲（打呼噜，有呼吸暂停风险）
```

### 2. 矛盾积累

```
袁利豪的困境（核心驱动）：
  - 与王立华合伙公司，王>50%控股，搞一言堂
  - 多次生意决策失败，袁利豪提意见不被理会
  - 袁利豪借钱开第二家公司 → 也亏损严重
  - 欠王立华大笔钱 → 王立华要求用股份抵债
  - 失去股份 = 失去东山再起机会 = 回到原点
  - 恳求宽限被拒 → 萌生杀意

丁姿的困境：
  - 王立华有打骂行为（"女人不打上房揭瓦"口头禅）
  - 提出分手 → 被王立华用偷拍的私密视频威胁
  - 不敢报警（网络传播的伤害无法弥补）
  - 被迫继续留在王立华身边，战战兢兢

袁利豪利用丁姿的过程：
  - 在事业最绝望时（3个月前）主动追求丁姿
  - 了解到丁姿的处境和视频要挟后，提出"帮她杀王立华"
  - 真实目的：让丁姿当枪手，自己脱身
  - 教丁姿杀人方法 + 提供安定药物
```

### 3. 犯罪决策与手段选择

```
袁利豪的策划：
  ① 周二接到许灿聚会邀请 → 利用这个机会
  ② 用专业手段（打孔技术）在啤酒瓶中灌入安定类药物
  ③ 啤酒是他带来的（做酒行生意，有技术条件）
  ④ 分发啤酒时，将有药的那瓶通过丁姿递给王立华
     （利用情侣间互递酒的习惯，而非直接递给王立华）
  ⑤ 自己全程不直接接触杀人行为

丁姿的执行：
  ① 王立华因酒精+安定药物陷入深度昏睡
  ② 丁姿借口"扶他上楼喝水"获得5分钟独处
  ③ 用矿泉水浸湿纸巾 → 贴加官手法 → 一张张贴在脸上
  ④ 5分钟内完成（药物昏迷+无挣扎）→ 下楼若无其事
  ⑤ 王立华在无意识中窒息死亡
  ⑥ 数小时后上楼 → 取走纸巾 → 扔出窗外 → 尖叫发现
```

### 4. 关键物证

```
① 死因: 缺氧引起的机械性窒息（法医确认）
② 面部暗红斑点: 口鼻周围，外力机械性窒息特征
③ 啤酒瓶中安定类药物: 某瓶含有安定药（技术检测）
④ 窗外纸巾团: 别墅后方植物枝杈上（陈益搜索发现）
⑤ 纸巾上矿物质残留: 匹配高端矿泉水成分（离子检测可证）
⑥ 纸巾上DNA: 王立华的皮屑脱落
⑦ 矿泉水瓶（空的在垃圾桶）: 但法医检测死者口腔无矿物质残留
   → 王立华没喝过那瓶水！→ 丁姿撒谎（水被用于浸湿纸巾）
⑧ 偷拍视频: 王立华电脑隐蔽文件夹中发现
⑨ 欠条: 王立华家中发现袁利豪的大额欠款
⑩ 公司财务: 袁利豪两家公司严重亏损
⑪ 录音: 丁姿手中保留的袁利豪策划杀人录音（丁姿后手）
```

### 5. 暴露点

```
暴露点A（安定药物 → 指向袁利豪）：
  - 啤酒瓶中检出安定类药物
  - 袁利豪做酒行生意 → 有打孔灌注技术
  - 袁利豪分发的啤酒
  发现条件: 毒理检测(技术力4+)

暴露点B（矿泉水谎言 → 指向丁姿）：
  - 丁姿说"王立华先喝了水再睡" → 但法医检测口腔无矿物质
  - 矿泉水被用于浸湿纸巾而非饮用
  发现条件: 法医学(离子检测) + 审讯矛盾捕捉

暴露点C（窗外纸巾 → 作案工具）：
  - 别墅后方发现湿纸巾团
  - 位置对应二楼窗户
  - 含矿物质残留+死者DNA
  发现条件: 现场搜索(观察力5+)

暴露点D（递酒顺序异常 → 袁利豪有预谋）：
  - 给王立华的酒经过丁姿递 → 但给姜凡磊直接站起来递
  - 不一致的行为暗示预谋
  发现条件: 记忆力 + 细节观察(观察力5+)

暴露点E（欠条+公司亏损 → 袁利豪深层动机）：
  - 借钱开第二家公司失败
  - 被要求用股份抵债
  - 追求丁姿的时机=事业最绝望时
  发现条件: 经侦调查 + 逻辑推理

暴露点F（录音证据 → 铁证合谋）：
  - 丁姿保留了袁利豪策划杀人的录音
  - 在互撕中被逼出
  发现条件: 审讯博弈(心理学5+) → 让两人互相揭发
```

## 二、游戏化设计要点

### 案件独特之处

**"密室推理+博弈心理"** — 不是传统密室，而是"所有人都在场，但凶手能在所有人眼皮底下完成杀人"。

关键机制：
1. **延时杀人**: 杀人行为(贴纸巾)和死亡结果分离，中间有数小时间隔
2. **双人合谋**: 一人提供工具+策划，另一人执行
3. **博弈互撕**: 案破后两人相互指控，需要判断谁说的真谁说的假
4. **多层伪装**: 最初看像意外→发现安定药→又发现真正的杀人手法

### 属性检定映射

| 调查行动 | 所需属性 | 阈值 | 获得信息 |
|---------|---------|------|---------|
| 判断死因是机械性窒息 | 法医学 3+ | 中 | 初步判断死因 |
| 发现面部暗红斑点意义 | 法医学 5+ | 高 | 外力窒息而非自然窒息 |
| 联想到鼻中隔偏曲→意外可能 | 知识(医学) 3+ | 中 | 提供意外假说 |
| 搜查别墅周围发现纸巾 | 观察力 5+ | 高 | 关键物证 |
| 记住递酒顺序并发现异常 | 观察力 5+ | 高 | 指向袁利豪预谋 |
| 质疑矿泉水→法医检测 | 逻辑推理 + 法医学 | 高 | 戳穿丁姿谎言 |
| 查袁利豪公司财务 | 经侦力 3+ | 中 | 发现深层动机 |
| 利用互撕逼出录音证据 | 审讯力 5+ + 心理学 5+ | 极高 | 铁证 |

### 红鲱鱼

```
红鲱鱼1：意外死亡
  - 鼻中隔偏曲 + 饮酒过量 → 睡眠中呼吸暂停
  → 排除：面部暗红斑点+安定药物+纸巾

红鲱鱼2：袁利豪单独作案（安定药物）
  - 安定药导致深度昏迷致死
  → 排除：安定药很难直接杀人，需要配合其他手段

红鲱鱼3：全员嫌疑（五人都有动机）
  - 许灿(商业竞争)、姜凡磊(情敌)、陈益(被嘲讽)
  → 排除：均缺乏充分动机，更缺乏作案条件

红鲱鱼4：厨师
  - 外来人员，有接触食物的机会
  → 排除：身份核实无问题，无法精准控制目标
```

### 推理板设计

```
第1层：排除意外
[面部暗红斑点] + [安定类药物] → "不是简单的意外死亡"

第2层：锁定工具提供者
[啤酒中有安定药] + [袁利豪做酒行+分发啤酒] → "袁利豪投药"

第3层：找到真正杀人手法
[矿泉水瓶空了但死者没喝] + [窗外纸巾团+矿物质残留]
→ "贴加官：湿纸巾贴面窒息"

第4层：确定真凶
[只有丁姿有5分钟独处时间] + [纸巾上有死者DNA]
→ "丁姿执行了贴加官"

第5层：揭露合谋
[袁利豪深层动机(欠款+公司危机)] + [追求时机=事业最绝望时]
→ "袁利豪策划，丁姿执行，合谋杀人"
```

## 五、案件模板价值

此案展示了**"博弈型案件"**的极高价值：

1. **延时杀人机制**: 行为和结果分离 → 制造"不可能犯罪"假象
2. **合谋+背叛**: 两人合作但各怀鬼胎 → 审讯中互相揭发
3. **冷知识杀人法**: "贴加官"这种冷门知识 → 增加推理乐趣
4. **多疑点并行**: 意外/安定药/贴加官三层叠加 → 需要逐层排除
5. **审讯博弈**: 判断两人互撕中谁说真话 → 高阶心理学应用

在roguelike中，这种案件适合**高难度模式**：
- 需要法医学+心理学+经侦多项属性达到较高水平
- 审讯博弈环节可以设计为"选择相信谁"的关键抉择
- "贴加官"知识可以作为**特殊知识书/天赋解锁**才能发现的线索
