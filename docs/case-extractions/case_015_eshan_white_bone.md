## 三、矛盾积累（核心前史）

1. **严鹏的堕落**：创业成功→被小三搞垮→妻离子散→流浪至山村
2. **恩将仇报**：被山村陈家收留→觊觎陈家15岁女儿→奸杀陈诗然（罗翎姐姐）
3. **灭口连杀**：13岁弟弟陈东亮试图救姐→被严鹏摔下山崖致死
4. **身份替代**：途中杀害好心人龚耀光→顶替其身份→收养龚耀光幼子
5. **家庭崩溃**：父亲郁郁而终→母亲精神失常→罗翎被送人

---

## 四、触发事件

罗翎成年后，通过接近富豪籍臧阳获取资源，历经多年、花费近千万、动用37名私家侦探，终于查到"龚耀光"就是严鹏。

---

## 五、犯罪决策

- **法律无法伸张正义**：时间久远、证据难以收集
- **极端仇恨驱动**：姐姐被奸杀+弟弟被摔死+父母一死一疯
- **选择最残忍方式**：万虫噬身，让严鹏在极度恐惧和痛苦中死去

## 六、手段选择（暴风雪山庄+白骨表演）

### 前期准备（数年）
| 准备 | 详情 |
|------|------|
| 接近籍臧阳 | 了解其"恋人妻"癖好→找钟木平结婚→成功接近 |
| 获取资源 | 籍臧阳给予金钱、地位、人脉、厄影云麓支配权 |
| 调查严鹏 | 37名私家侦探、花费千万 |
| 改造庄园 | 在洗手间安装指纹识别暗门+密室 |
| 布置虫箱 | 利用籍臧阳的毒虫收藏作为杀人工具 |
| 安排观众 | 主动结识龚蔚帆（真龚耀光侄女）引至庄园 |
| 让籍臧阳邀请"龚耀光" | 让目标主动进入杀局 |

### 案发执行
1. 暴雪封山，九人被困厄影云麓
2. 深夜，曲林江用钥匙进入"龚耀光"房间迷晕他
3. 罗翎用绳索+滑轮从窗户将其拉至四楼密室
4. 放入装满毒虫（蛇、蜈蚣、蝎子、蜘蛛等）的木箱中钉死
5. 将真人白骨（姐姐的遗骨，涂上仿模型材料）放在"龚耀光"床上
6. 罗翎通过暗门藏入密室，制造"失踪"假象

### 白骨表演设计
- 客厅骨骼模型"消失"→引发搜查→发现"龚耀光"房间有白骨→呼应厄山大妖传说
- 白骨亲子鉴定与龚耀光之子吻合（因为白骨是龚耀光的同卵双胞胎姐姐）

## 八、游戏设计价值

### 核心设计亮点
1. **暴风雪山庄模式**：完美的封闭推理关卡——暴雪封山、信号隔绝、九人局
2. **白骨传说+真实白骨**：虚实结合，传说为案件提供恐怖氛围和误导
3. **身份替代双层结构**：假龚耀光=严鹏（凶手变身）+ 假陈诗然=罗翎（复仇者变身）
4. **九人狼人杀**：每个人都有秘密，玩家需要在社交中识别真实意图
5. **毒虫杀人法**：极具视觉冲击力和恐怖感的杀人手法
6. **人际博弈**：权力、金钱、爱情、复仇的多维交织
7. **灰色道德**：严鹏死有余辜 vs 罗翎手段残忍

### 可复用元素
- **暴风雪山庄框架**：封闭空间+有限嫌疑人+无法求援
- **身份替代谜题**：某个角色不是他自称的人
- **密室消失术**：暗门+指纹识别的现代密室
- **传说与真相**：地方传说作为案件伪装
- **观察力挑战**：微表情、身高差异、材质差异

## 十、红鲱鱼/误导设计

1. **厄山大妖传说**：白骨=大妖吞噬血肉→实为人为布置
2. **骨骼模型"被偷"**：引导怀疑偷窃→实为真人白骨调包
3. **龚耀光的"认亲"**：龚蔚帆认亲→观众以为是巧合→实为刻意安排
4. **陈诗然的"失踪"**：仿佛也被大妖所害→实为藏入密室
5. **亲子鉴定吻合**：白骨DNA与龚耀光之子匹配→不是龚耀光而是其双胞胎
6. **钟木平的"交友"目的**：表面交朋友→实为借刀杀人

## 十四、特殊备注

此案是全书最适合改编为游戏关卡的案件之一：

1. **暴风雪山庄=完美的roguelike关卡**：封闭空间、有限NPC、无法求援、时间压力
2. **九人局狼人杀机制**：玩家需要通过对话和观察识别每个人的真实身份和目的
3. **白骨传说=环境叙事**：厄山大妖传说为案件增加恐怖氛围层
4. **密室搜索+物理谜题**：指纹暗门、滑轮系统、虫箱
5. **多层身份欺骗**：假龚耀光=严鹏 + 假陈诗然=罗翎，身份替代作为核心谜题
6. **高端人际博弈**：权力、金钱、利用、复仇交织的复杂人际网
7. **弩箭QTE**：陈益接弩箭的场景可设计为动作检定环节
