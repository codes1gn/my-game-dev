## 三、矛盾积累

### 第一层：不育悲剧
- 宋立顺天生不育，但这一秘密被妻子们掌控
- 前妻陶培娟迫于压力与小叔子宋立国通奸，生下宋平全
- 宋立顺毫不知情，将宋平全视为亲子抚养

### 第二层：续弦出轨
- 付蓉蓉嫁入豪门为求富贵，非出于爱情
- 与继侄宋平辉通奸生下王宝贵
- 通过检查获知宋立顺不育真相，却隐瞒不报
- 又与继子宋平全发展关系

### 第三层：遗产阴谋
- 宋平全身世被家族多人知晓
- 王宝贵出生威胁遗产格局
- 家族成员合谋偷走王宝贵（婴儿失踪案）
- 宋立顺多年为他人养子、积累上亿财富最终为他人做嫁衣

## 四、触发事件

宋立顺尿毒症晚期 → 全家配型失败（因无亲生子女） → 寻找"失散儿子" → 网络视频意外发现王宝贵 → 亲子鉴定：非亲子但有亲缘 → 医生解读引发怀疑

**关键转折**：DNA亲子鉴定技术无意间揭开了二十年家族骗局

## 五、犯罪决策

### 决策过程
1. 得知王宝贵非亲生 → 初始怒火指向付蓉蓉
2. 强行摘肾保命 → 怒火暂压
3. 术后第5天逼供付蓉蓉 → 连锁爆料：
   - 王宝贵是宋平辉之子
   - 宋平全也非亲生（宋立国之子）
   - 自己天生不育
   - 全家在等自己死后分财产
4. 七十年人生崩塌 → 决定"都该死"

### 心理剖析
- **极端自尊**：一辈子的家族权威被践踏
- **丧子之痛的反转**：以为有两个儿子，实际一个没有
- **被背叛的绝望**：两任妻子、两个弟弟、两个侄子全部背叛
- **报复心理**：不是简单泄愤，而是"绝不能让他们得逞"

## 六、手段选择与执行

### 第一案：杀害付蓉蓉（术后第5天）
- **方式**：在养殖场机械厂区逼问后杀害
- **毁尸**：饲料粉碎机碎尸 → 喂养牲畜
- **善后**：更换血衣、烧毁证据

### 第二案：毒杀全家（术后第11天）
- **预谋**：给养殖场员工放假三天（清场）
- **毒物**：氟乙酸（灭鼠剂主成分，养殖场现有）
- **投毒方式**：提前混入全部厨房佐料（盐/醋/酱油/油）+ 酒瓶
- **自保**：术后不能饮酒、食量极少 → 中毒剂量远低于他人
- **以身入局**：自己也中毒被送医 → 伪装为受害者

## 九、知识标注

### 法医/刑侦技术
- **鲁米诺反应**：检测隐藏血迹，铁离子在紫外线下发淡蓝色荧光
- **DNA亲子鉴定vs亲缘鉴定**：亲子=直系父母子女；亲缘=旁系血缘关系
- **痕迹学**：脚印覆盖比对、指纹排除法
- **紫外线痕迹灯**：波长可调，用于犯罪现场血迹检测

### 毒理学
- **氟乙酸**：有机剧毒化合物，灭鼠剂主成分
  - 性状：无色/白色结晶粉末
  - 溶解性：易溶于水、易溶于乙醇
  - 中毒症状：呕吐→腹泻→麻木→癫痫→呼吸衰竭→心搏骤停
  - 致死机制：损害中枢神经和心脏

### 医学知识
- **肾脏移植配型**：直系亲属成功率~90%+，旁系亲属也有较高概率
- **不育症**：可导致数十年隐瞒的家庭悲剧
- **术后恢复**：肾脏移植术后通常需较长住院观察

### 法律知识
- **组织出卖人体器官罪**：五年以下，情节严重五年以上
- **故意伤害罪**（强摘器官）
- **第一顺位继承人**：配偶、子女、父母
- **公诉案件**：受害者不追究仍会立案

### 社会学
- **县域富商生态**：板材+养殖产业、一体化运营
- **老夫少妻**：年龄差距大的婚姻动机分析
- **宗族取名传统**："立"字辈、"平"字辈
