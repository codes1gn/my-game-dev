## ===== 案件20A：苟其文案（拆迁户伪装自杀） =====

### 一、因果链总览

```
[背景关系网] 苟其文（二流子/拆迁户）→ 苟军勇（同村本家/追同一个女孩失败）
      ↓
[矛盾积累] 苟其文抢走了苟军勇追求的女朋友 → 苟军勇内心积怨
      ↓
[触发事件] 苟其文与开发商拆迁纠纷 → 苟其文计划自导自演"假上吊"逼迫开发商就范
          → 凌晨电话约苟军勇来帮忙拍照/配合演戏
      ↓
[犯罪决策] 苟军勇看到苟其文主动将脖子套入绳圈 → 机会出现 → 一脚踢翻凳子
      ↓
[执行过程] 苟其文瞬间吊死（几秒钟内脑缺氧无法自救）→ 苟军勇离开
      ↓
[暴露点] 调研组程汉君发现凳子距离不对（脚印靠后但凳子飞出过远）→ 实验证明无法复现
        → 最后通话人苟军勇 → 动机：抢女人之恨
```

### 二、核心侦查技巧

1. **脚印位置+凳子飞行距离实验**：程汉君敏锐发现脚印位于凳子边缘靠后，多次实验证明无法将凳子踢到案发位置
2. **洛卡德物质交换原理**：嫌疑人鞋面踢凳子→物质交换→锁定唯一嫌疑人
3. **自杀动机不成立分析**：马上拆迁的人为何自杀？"一夜暴富"的人自杀动机根本不存在

## ===== 案件20B+C：四女连环强奸案 + 钟落村杀人案 =====

### 一、因果链总览（核心因果链 - 两案合一）

```
[背景关系网] 钟落村：偏远山村/有山葬习俗/年轻人极少
            → 刘诚朴（老实本分村民）+ 刘首乌/刘针茅（狡猾同龄人/三人常一起上山）
            → 丁云洁（邑城大学登山社女生/内向柔弱/后来改变性格）
            → 何珊+江丽丽+任丹（丁云洁大学好友/登山社成员）
      ↓
[矛盾积累·隐藏真相] 何珊四人登山到钟落山 → 刘首乌和刘针茅在山上强奸了丁云洁
                    → 丁云洁对三个朋友隐瞒了被强奸事实
                    → 刘诚朴喜欢丁云洁 → 得知强奸事实后愤怒至极
      ↓
[触发事件] 丁云洁利用刘诚朴对自己的感情 → 指使/教唆刘诚朴杀害刘首乌和刘针茅
          → 刘诚朴用商陆（致昏迷的山药材）将二人迷晕 → 背到葬谷扔下喂狼
          → 烧毁案发相关的山中木屋（毁灭证据）
      ↓
[犯罪升级] 丁云洁继续利用刘诚朴 → 教唆他强奸自己的三个朋友作为报复
          → 何珊是主要目标（性格强势毒舌/丁云洁最恨的人）
          → 丁云洁教刘诚朴反侦察技巧（前三人不留痕迹）
          → 何珊被留到最后：不教反侦察→留下DNA→毁容→故意让警方能查到
      ↓
[执行过程] 刘诚朴下山进城 → 找到四人住址 →
          ①强奸江丽丽（夜跑时打晕）→ ②强奸任丹（尾随入户打晕）
          → ③强奸丁云洁（酒吧醉倒后/实为配合演戏）→ ④虐奸何珊（车库打晕/毁容/留DNA）
      ↓
[善后行为] 丁云洁主导"受害者联盟"→说服任丹不报警→自己也假装受害者
          → 前三人未报警/丁云洁第三个报警（做样子）→ 何珊去江城后终于报警
      ↓
[暴露点] 何珊江城报案 → 调研组介入 → DNA全面采集 → 比对锁定钟落村刘诚朴
        → 陈益在葬谷附近发现平安符（丁云洁赠送给刘诚朴）→ 揭示二人关系
        → 陈益利用小说心理审讯 + 父母亲情攻心 → 刘诚朴道心破碎 → 交出丁云洁
```

### 二、关键人物

| 角色 | 身份 | 核心特征 |
|------|------|----------|
| 刘诚朴 | 执行者/工具人 | 钟落村老实村民、外卖骑手、深爱丁云洁、为爱扛下所有罪 |
| 丁云洁 | 幕后主谋/教唆者 | 内向→性格巨变、网络作家、被刘首乌等强奸后策划复仇、极度隐忍 |

| 刘首乌/刘针茅 | 被杀者/初始施暴者 | 钟落村狡猾青年、强奸丁云洁→被刘诚朴喂狼 |

### 三、核心侦查技巧（游戏设计价值）

1. **DNA全面采集大网**：对受害者所有认识的人进行DNA采集→大海捞针式排查
2. **平安符线索**：葬谷丛林中找到被狐狸叼走的平安符→玫瑰绣花→揭示刘诚朴有"喜欢的女孩"
3. **山药材商陆**：形似人参但含生物碱→致昏迷→森林公安提供的药材知识
4. **小说心理审讯**：陈益读完丁云洁的网络小说→用书中角色类比刘诚朴→"柏向峰式的牺牲毫无意义"
5. **亲情攻心**：弹弓（父亲的爱）+ 父母探视 → 道心破碎 → 嫌疑人彻底崩溃认罪
6. **"三次强奸不留痕迹，第四次故意留下"**：丁云洁教唆的证据——前三次反侦察/最后一次放飞
7. **伪装受害者**：丁云洁把自己也安排成"第三个受害者"→灯下黑→成功隐藏教唆者身份
8. **钟落村山葬风俗**：葬谷喂狼→完美毁尸手段→利用当地风俗作为杀人工具

### 四、深层主题

- **工具人的爱与牺牲**：刘诚朴为丁云洁甘愿赴死→丁云洁利用了他的爱→"你什么都不算"
- **被害者变施害者**：丁云洁从强奸受害者变成连环犯罪教唆者——痛苦转化为仇恨
- **性格决定命运**：何珊的毒舌→被丁云洁最恨→被毁容；刘诚朴的老实→被利用为杀人工具
- **山村与城市的碰撞**：钟落村的封闭原始 vs 城市的法治文明→两种世界的悲剧交汇
- **隐忍的代价**：丁云洁不报警不求助→自行复仇→最终伤害更多人

### 五、钟落村世界观（游戏设计素材）

- **山葬习俗**：老人临终前食用商陆昏迷→被背到葬谷→被狼群分食（自然葬法）
- **吉君制度**：村中有"吉君"角色（类似巫师/族老），维持传统风俗
- **封闭社会**：村民极少下山，对外界几乎无了解，法治观念淡薄
- **自然危险**：野狼出没的葬谷、商陆致昏迷、拉拉秧划伤——自然环境即凶器

- **林辰**：无直接参与（留在调研工作中）
- **傅国勇**：实验辅助（最大力气踢凳子验证）

## 七、游戏设计亮点提取

1. **伪装自杀机制**：玩家需分析脚印/凳子距离/实验复现来判断自杀真伪
2. **DNA全面采集系统**：大范围排查→每个NPC都有DNA→比对成功即破案
3. **山村探索**：钟落村的封闭世界——独特风俗、危险环境、隐藏线索（平安符/木屋灰烬）
4. **心理审讯mini-game**：利用嫌疑人的弱点（小说角色/童年弹弓/父母探视）逐步击溃防线
5. **灯下黑机制**：受害者中有一个是幕后黑手——玩家需从四个"受害者"中识别教唆者
6. **跨案联动**：登山→杀人→强奸→三个看似无关的案件通过一个平安符串联
