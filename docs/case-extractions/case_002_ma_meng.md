## 一、完整因果链

### 1. 背景关系网

```
关系类型：极复杂的三角+雇凶+敲诈多层网络

付林旺 ←(丈夫/妻子, 结婚2年)→ 马萌(死者)
付林旺 ←(前夫/前妻, 结婚近10年)→ 杨芳萍
范世海 ←(前男友/前女友, 马萌入狱前恋人)→ 马萌
范世海 ←(送货员/老板, 范世海为接近马萌而应聘)→ 付林旺
范世海 ←(狱友)→ 韩家光(真凶)
韩家光 ←(敲诈者/被敲诈者)→ 马萌

关键背景：
- 付林旺: 产品代理, 年入几十万到百万, 二婚, 经常出差
- 马萌: 前夜场工作者(有卖淫案底), 转型网络主播, 年轻貌美
- 范世海: 有前科(聚众斗殴3年), 送货员, 对马萌感情很深
- 韩家光: 有前科(非法持有毒品), 汽车修理工, 出狱后仍吸毒
- 杨芳萍: 保险经纪人, 付林旺前妻, 干练强势
- 付林旺患不孕不育(输精管梗阻, 4年前确诊, 隐瞒)
```

### 2. 矛盾积累（多条并行）

```
矛盾线A（付林旺→马萌）：
  - 付林旺经常出差 → 马萌不满
  - 付林旺发现马萌怀孕化验单 → 自己不孕不育 → 马萌出轨
  - 好面子+自尊心强+小心眼 → 愤怒到极点

矛盾线B（范世海→马萌）：
  - 范世海入狱 → 马萌没等他, 嫁给了付林旺
  - 出狱后不甘心 → 刻意应聘付林旺送货员 → 接近马萌
  - 趁付林旺出差 → 和马萌旧情复燃

矛盾线C（韩家光→马萌）：
  - 范世海酒后向狱友韩家光炫耀马萌照片+出轨信息
  - 韩家光起邪念 → 用出轨信息威胁马萌 → 求财+求色
  - 马萌被迫就范 → 多次向付林旺要钱(用于给韩家光)
  - 马萌发现韩家光仍吸毒+持有毒品 → 反要挟
  - 韩家光被反制 → 暴怒 → 决定杀人灭口
```

### 3. 两条独立的杀人动机（平行存在）

```
付林旺的雇凶计划（未遂）：
  触发: 发现马萌怀孕化验单 → 确认出轨
  决策: 自己出差制造不在场证明 → 雇人杀妻
  选择范世海: 有前科→可能愿意、手头紧→需要钱、是自己手下
  执行: 取现金20万 → 第二天出差 → 等待结果
  结果: 范世海收钱但没杀（因为他就是出轨对象）

韩家光的杀人计划（成功）：
  触发: 马萌用毒品信息反要挟 → 韩家光感到失控
  决策: 杀人灭口
  手段: 利用与马萌的"熟人"关系敲门入室
  执行: 马萌穿睡衣开门 → 从背后刺杀 → 洗劫财物(伪装抢劫) → 清理脚印
  反侦察: 戴手套(口中留下纤维)、清理脚印、穿冲锋衣帽子口罩
```

### 4. 执行过程与物证

```
物证清单：
① 尸体: 两处贯穿刀伤(背部), 致命伤在心脏
② 手套纤维: 死者口腔内(凶手捂嘴时留下)
③ 现场被翻乱: 伪装抢劫, 贵重物品+手机被带走
④ 脚印被清理: 只有死者和付林旺的脚印
⑤ 监控录像: 拍到穿冲锋衣+帽子+口罩的男性, 身高可辨
⑥ 怀孕化验单: 藏在书房柜子书中(隐秘位置)
⑦ 银行取款记录: 付林旺上周日取现金20万
⑧ 20万现金: 在范世海酒店房间包中发现
⑨ 通话记录: 马萌&付林旺都与范世海频繁联系
⑩ 凶器匕首: 韩家光随身携带(持刀拒捕时暴露)
```

### 5. 暴露点与调查路径

```
暴露点A（心理矛盾 → 指向付林旺有问题）：
  - 付林旺对"为何先报警不叫救护车"的回答过于完美
  - 情绪只有悲伤没有愤怒 → 与其性格不符
  发现条件: 审讯经验 + 犯罪心理学 3+

暴露点B（离婚原因 → 揭示不孕不育）：
  - 付林旺说"感情不和" → 杨芳萍说"因为孩子"
  - 杨芳萍无问题 → 付林旺有不孕症 → 马萌怀孕=出轨
  发现条件: 走访技巧(魅力3+) + 逻辑推理

暴露点C（20万取款 → 雇凶证据）：
  - 取现金20万 → 第二天出差 → 然后马萌被杀
  - 说"丢了" → 漏洞百出但无直接反驳证据
  发现条件: 经侦/财务追踪(技术力3+)

暴露点D（化验单指纹 → 逼付林旺承认知情）：
  - 化验单上有付林旺指纹 → 他早就知道马萌怀孕
  发现条件: 痕检(指纹提取) + 审讯技巧

暴露点E（共同人际关系 → 指向范世海）：
  - 马萌和付林旺通话记录中都有范世海
  - 微信聊天暧昧 + 电话频繁 → 出轨对象
  发现条件: 技侦(通话/微信分析)

暴露点F（身高差异 → 排除范世海, 指向另一个人）：
  - 监控身高与范世海差2cm → 不是凶手
  - 范世海的狱友韩家光身高1.75完美匹配
  发现条件: 观察力 5+(身高测算) + 逻辑推理

暴露点G（敲诈动机链 → 指向韩家光）：
  - 范世海酒后炫耀 → 韩家光得知马萌信息
  - 马萌多次向付林旺要钱(被韩家光敲诈)
  - 马萌发现韩家光持毒反要挟 → 韩家光杀人灭口
  发现条件: 犯罪心理学 5+ + 审讯突破范世海

暴露点H（韩家光逃跑 → 实锤有罪）：
  - 见到警察立即逃跑 → 不是毒品就是杀人

  发现条件: 抓捕行动(体能5+)
```

## 二、游戏化设计要点

### 案件独特之处

这个案件的核心魅力在于**三层嵌套**：
1. 表面看是入室抢劫杀人
2. 深入发现是丈夫雇凶杀妻
3. 但雇的人没杀, 真凶是另一个方向的敲诈者

玩家可能在第2层就以为破案了，实际上需要到第3层才真正找到凶手。

### 属性检定映射

| 调查行动 | 所需属性 | 阈值 | 获得信息 |
|---------|---------|------|---------|
| 审讯付林旺(初次) | 审讯力 3+ | 低 | 发现回答可疑 |
| 走访杨芳萍 | 魅力 3+ | 中 | 离婚原因=孩子问题 |
| 查付林旺病历 | 技术力 2+ | 低 | 确认不孕症 |
| 查银行取款记录 | 技术力 3+ | 中 | 发现20万取现 |
| 逼问化验单指纹 | 审讯力 5+ | 高 | 付林旺承认知情 |
| 分析通话记录重合 | 技术力 3+ | 中 | 范世海=共同联系人 |
| 判断监控身高差异 | 观察力 5+ | 高 | 排除范世海 |
| 推断狱友关系 | 心理学 5+ | 高 | 引出韩家光 |
| 追踪韩家光位置 | 技术力 5+ | 高 | 身份证监控定位 |
| 抓捕韩家光 | 体能 5+ | 高 | 完成逮捕 |

### 红鲱鱼

```
红鲱鱼1：付林旺=凶手
  - 有动机(马萌出轨)、有行为(取20万)、有性格(小心眼)
  → 排除：有不在场证明(江城出差, 监控+客户证实)
  → 但：确实雇了凶(只是雇的人没杀)

红鲱鱼2：范世海=杀手
  - 收了20万、有前科、是马萌出轨对象
  → 排除：身高差2cm + 嫖娼不在场 + 收钱没杀(诈骗)
  
红鲱鱼3：入室抢劫
  - 现场翻乱、财物失踪
  → 排除：是凶手伪装(掩盖真实的杀人灭口动机)
```

### 推理板设计

```
第1层推理：
[背后刺杀+穿睡衣] → "熟人作案, 死者无防备"
[怀孕化验单+不孕症] → "马萌出轨"
[20万取现+第二天出差] → "付林旺雇凶杀人"

第2层推理（推翻第1层结论）：
[范世海=共同联系人] + [范世海=马萌前男友] → "范世海是出轨对象"
[范世海收了钱] + [身高差2cm] → "范世海没杀, 他是诈骗"
→ "真凶另有其人"

第3层推理：
[凶手是熟人] + [有反侦察] + [范世海的狱友?] → "范世海的某个联系人"
[韩家光身高匹配] + [非法持毒前科] → "韩家光有嫌疑"
[马萌反复要钱] + [韩家光知道马萌出轨] → "敲诈→反要挟→灭口"
```

## 五、案件模板价值

此案件展示了一种**极具roguelike价值的结构**：

**"洋葱型案件"** — 每剥一层都以为到底了，但下面还有一层。

关键机制：
- 第1层答案（入室抢劫）→ 错误，但看似合理
- 第2层答案（雇凶杀人）→ 部分正确（确实有雇凶），但凶手不对
- 第3层答案（敲诈灭口）→ 真相

这种结构让玩家体验到"推翻自己之前的结论"的快感，而不是简单的"选对答案"。
每一层的推翻都需要新的证据和更深的推理。
